## March 4

- [Warehouse (PyPI)](https://warehouse.pypa.io/development/getting-started/#logging-in-to-warehouse) a great example of OSS project with clear installation steps, making it super easy to contribute
- [chezmoi](https://github.com/twpayne/chezmoi) a tool for managing your dotfiles securely across multiple machines
- [git-sync](https://github.com/simonthum/git-sync) a bash script that automatically synchronizes tracking repositories by committing, pulling, and pushing changes via a central git repository
- [no-quarantine](https://github.com/mendymm/no-quarantine/) a Rust process that monitors your Downloads folder and automatically removes the macOS quarantine attribute from newly downloaded files

- [Dwitter](https://www.dwitter.net/) a platform for creating and sharing JavaScript animations in 140 characters or less
- [Xikipedia](https://xikipedia.org/) a pseudo social media feed that algorithmically curates content from Simple Wikipedia based on your engagement patterns, all processed locally on your device
- [Times New Resistance](https://kottke.org/26/02/times-new-resistance) a typeface that looks like Times New Roman but autocorrects specific words to political statements
- [Pope Priests AI](https://futurism.com/artificial-intelligence/pope-priests-ai) Pope Leo XIV urged Catholic priests to stop using ChatGPT to write sermons, arguing that AI cannot replicate the human capacity to share faith
- [Snake](https://snake.eieio.games) a protocol and client for securely playing multiplayer snake over the internet via SSH
- [WebVM](https://webvm.io/) Linux virtualization in WebAssembly, running a complete Linux environment directly in the browser without a server
- [TLDR Newsletter](https://tldr.tech/) a free daily email with summaries of the most interesting stories in startups, tech, and programming
- [tldr-pages](https://github.com/tldr-pages/tldr) collaborative cheat sheets for console commands, and the in-browser version [tldr.inbrowser.app](https://tldr.inbrowser.app/)
- [DuckDuckGo](https://duckduckgo.com/) which can also directly search on tldr with the `!tldr` shortcut
- [Duck.ai](https://duck.ai/?origin=funnel_home_website) a free, private AI chat service provided by DuckDuckGo
- 📕 [The Art of Computer Programming](https://en.wikipedia.org/wiki/The_Art_of_Computer_Programming) series of books by Donald Knuth, a comprehensive monograph on computer programming algorithms and their analysis
- 📕 [ast-grep](https://leanpub.com/ast-grep) a book on using Abstract Syntax Trees for structural code searching, linting, and refactoring
